# Fonts Directory

Ce dossier est destiné aux fichiers de polices personnalisées.

## Polices Poppins requises

Pour activer les polices Poppins dans l'application, ajoutez les fichiers suivants :

- `Poppins-Regular.ttf`
- `Poppins-Medium.ttf`
- `Poppins-SemiBold.ttf`
- `Poppins-Bold.ttf`

## Activation

Une fois les fichiers ajoutés :

1. Décommentez les lignes dans `src/shared/theme/typography.ts`
2. Décommentez les lignes dans `app/_layout.tsx`
3. L'application utilisera automatiquement les polices Poppins

## Téléchargement

Les polices Poppins peuvent être téléchargées depuis :
- Google Fonts : https://fonts.google.com/specimen/Poppins
- Font Squirrel : https://www.fontsquirrel.com/fonts/poppins
